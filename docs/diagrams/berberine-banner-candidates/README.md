# Berberine Banner Candidates

Public-safe visual candidates for BioSymphony GeneCluster README, social preview, or release-note experiments.

## Files

- `berberine-02-hero.png` is the strongest banner-shaped hero candidate (`1693 x 929`).
- `berberine-01-campaign.png`, `berberine-02-pipeline.png`, and `berberine-03-results.png` are structured `1600 x 900` panels.
- `berberine-01-hero.png` and `berberine-03-hero.png` are portrait hero covers (`1024 x 1536`).

## Use Notes

- These are raster PNGs with metadata stripped before import.
- Treat the results panel as demo-specific copy. Review visible claims before using it as a generic repository banner.
- Keep future variants public-safe: no private run logs, provider IDs, raw/heavy biological data, or unpublished sequences.
