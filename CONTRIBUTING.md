# Contributing

BioSymphony GeneCluster is a public-safe control-plane repo. Contributions should improve reusable contracts, validators, examples, docs, or public-safe workflow scaffolding.

Before opening a change:

- Keep raw reads, private sequences, generated heavy outputs, provider logs, and credentials out of the repo.
- Use public, synthetic, or placeholder data in examples.
- Run `make public-audit`.
- Keep claims bounded to what the included public evidence actually supports.

Provider examples should use environment variable names and placeholder paths only.
